## ربط المشروع بـ GitHub

ربط GitHub لا يتم عبر تعديل الكود — بل من واجهة Lovable مباشرة.

### الخطوات
1. افتح قائمة **(+)** في أسفل يسار مربع الدردشة
2. اختر **GitHub → Connect project**
3. فوّض تطبيق Lovable على GitHub
4. اختر الحساب أو المنظمة
5. اضغط **Create Repository**

### إذا لم يعمل الربط، تحقق من:
- حساب GitHub غير مربوط بحساب Lovable آخر (يُسمح بحساب واحد فقط)
- منح صلاحيات كافية لتطبيق Lovable على GitHub
- عدم وجود مستودع بنفس الاسم مسبقاً

### ملاحظة
لا يمكنني تنفيذ الربط من جهتي — يجب أن تقوم به من الواجهة. إذا ظهرت رسالة خطأ محددة، أرسل لي لقطة شاشة أو نص الخطأ وسأساعدك في تشخيصها.
